Import Token Sesi Fasih-SM

Login ke fasih-sm.bps.go.id di browser, lalu buka DevTools (F12) dan ambil:

  • Bearer: tab Application → Local Storage → key token
  • CSRF: tab Application → Cookies → XSRF-TOKEN (URL-decoded)
  • Cookies: tab Network → pilih request apapun ke fasih-sm.bps.go.id → header Cookie
  • User Agent: Console → ketik navigator.userAgent
